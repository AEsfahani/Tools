How to works:

 

Step 1: Creating a Comparison File
To create a comparison file perform the following:

 

1.      Click on set comparison folder, this folder would be the clients folder that you want to create an XML list of all of their files

2.      After selecting the folder click on Create Comparison File button

a.       Set a file location for the output.

b.      A temporary file name will be suggested. This comparison file will be used later on.

c.       Press the Save button

d.      Notice the comparison file field is now set to your newly created file.

Step 2: Setting the Base File or choose one of base file for Sage 100 versions

To set the base file perform the following:

1.       Click the Set Base File button

2.       Assign a base XML file for the basis of comparison

To select the base file from list just select the version you like to compare to from the drop down.

 

Step 3: Reviewing the results

To compare the two files you have defined in the comparison field and base field

1.       After the base file and the comparison file have been set, click on the comparison tab or click on the Go to comparison button

2.       Select your desire view setting

3.       Click on compare files button, you will see the results in the grid.

 

Note: You can sort the columns same way you can sort columns in excel sheet. The row state determines if the file has changed and any state other than “unchanged” represents a difference between the comparison file and base file.

 

Note: You can also search for a specific file using the search field. 

 

The Row State identifies the state of individual files contained in the scan. Any file in a state other than “Unchanged” is indicative of a customization or deviation from standard.
